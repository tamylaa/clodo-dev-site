Frequently Asked Questions

Find answers to common questions about Clodo Framework, Cloudflare Workers, and building serverless applications.

General

+ What is Clodo Framework?

Clodo Framework is a modern SaaS development framework that runs on Cloudflare Workers. It provides service templates, middleware, and utilities to help you build production-ready applications in weeks instead of months.

+ Do I need to know Cloudflare to use Clodo?

No. Clodo abstracts away most Cloudflare complexity. You can write standard Node.js/TypeScript code and Clodo handles the edge deployment details. Of course, learning Cloudflare concepts helps optimize your applications.

+ Is Clodo open source?

Yes! Clodo Framework is open source on GitHub. You can inspect the code, contribute, and use it for commercial projects.

+ Can I use Clodo with my existing project?

Yes. You can adopt Clodo incrementally. Start by migrating one service and gradually move more functionality to Clodo as you become comfortable with it.

Pricing & Costs

+ How much does Clodo Framework cost?

Clodo Framework itself is free and open source. You only pay for Cloudflare resources you use: $0.50 per 1M CPU milliseconds and $0.15 per 1M requests beyond the free tier.

+ Is there a free tier?

Yes! Cloudflare offers a generous free tier: 100,000 requests/day, 10ms CPU time per request, and 1GB KV storage. This is enough for testing and small projects.

+ How does pricing compare to Lambda/other serverless?

Clodo on Cloudflare is typically 5-10x cheaper than Lambda for similar workloads because: (1) you only pay per request, not per 100ms chunk, (2) no cold starts or warm-up costs, (3) global deployment is included.

+ What if my app grows to millions of requests?

Great question! At 10M requests/day: Lambda costs $500-1000/month, Clodo costs $20-50/month. The cost advantage scales with usage.

Performance

+ How fast is Clodo?

Response times are typically 10-50ms (P95). This includes database queries. Compare to Lambda: 300-500ms average.

+ Does Clodo have cold starts?

No. Cloudflare Workers are always warm and ready to respond immediately. No cold start penalty.

+ Can I use Clodo for real-time applications?

Yes! Clodo includes Durable Objects for stateful real-time applications like collaborative editing, multiplayer games, and live chat.

+ What is the maximum request size?

Clodo supports requests up to 100MB in size, with typical payloads being much smaller.

Security

+ Is Clodo secure for production?

Yes. Clodo is production-ready and used by many companies. You get Cloudflare's built-in DDoS protection, WAF, and security features at no extra cost.

+ How do I store sensitive data like API keys?

Use Cloudflare Workers Secrets. Secrets are encrypted and injected at runtime. They're never stored in your source code or logs.

+ Can I use Clodo with GDPR/CCPA compliance requirements?

Yes. You can process data at the edge without sending to a centralized data center. This simplifies data residency requirements. Cloudflare also offers GDPR-compliant data handling.

+ How does Clodo prevent SQL injection?

Clodo encourages parameterized queries. When using D1 database, always use placeholders (?) instead of string concatenation: db.prepare("SELECT * FROM users WHERE id = ?").bind(userId).run()

Database & Storage

+ What databases does Clodo support?

Clodo includes D1 (SQLite) by default. You can also connect to PostgreSQL, MySQL, or any external database. D1 is recommended for new projects.

+ Can I use MongoDB with Clodo?

Yes! You can connect to MongoDB Atlas from Clodo workers. However, D1 (SQL) is simpler and recommended for most projects.

+ How much data can I store in D1?

D1 databases can store up to 1TB of data. This is plenty for most applications. You can create multiple databases if needed.

+ What is KV and when should I use it?

KV is a globally distributed key-value store. Use it for caching, sessions, or fast lookups. It has eventual consistency (KV is best-effort). Use D1 when you need strong consistency.

+ How do I backup my D1 database?

Cloudflare provides automated backups. You can also export your database with: wrangler d1 execute mydb --file backup.sql

Deployment

+ How do I deploy a Clodo application?

Simply run: wrangler deploy. Your code is deployed globally to 300+ Cloudflare data centers in seconds. No infrastructure setup needed.

+ Can I use my own domain?

Yes. You can use your own domain with Clodo. Point your domain to Cloudflare nameservers and Clodo will serve it.

+ How do I deploy to staging/production?

Use environments in wrangler.toml: wrangler deploy --env production deploys to production, wrangler deploy --env staging deploys to staging. Each environment has separate configuration and database.

+ Can I do zero-downtime deployments?

Yes! Cloudflare handles this automatically. New code is deployed to all locations simultaneously, and existing requests complete with old code.

Development

+ What languages does Clodo support?

Primarily TypeScript and JavaScript. Both run on the same V8 engine. TypeScript is recommended for type safety.

+ Can I use npm packages with Clodo?

Yes! Most npm packages work fine. Packages are bundled with your worker code. Avoid large packages that add significant bundle size.

+ How do I debug Clodo applications?

Use wrangler dev for local development and debugging. Use wrangler tail to view production logs in real-time.

+ Can I use async/await in Clodo?

Yes, absolutely. All Clodo worker code is async by default. Use async/await for clean, readable code.

+ Is there a maximum code size?

The uncompressed code size limit is 1MB, compressed. Most applications are well under this. If you exceed it, consider splitting into multiple workers or reducing dependencies.

Comparison

+ How does Clodo compare to Lambda?

Clodo is faster (10ms vs 300ms+), cheaper ($0.50/M cpu-ms vs $0.000001667/ms), and simpler (no cold starts, no configuration). Lambda has more mature ecosystem. Clodo is better for APIs and SaaS.

+ Clodo vs Next.js?

Next.js is a framework for full-stack apps with Node.js backend. Clodo is a framework for edge functions. You could use both: Next.js for frontend, Clodo for API layer.

+ Can I use Clodo with Vercel/Netlify?

Yes! You can deploy Next.js/frontend to Vercel and backend API to Clodo. This gives you the best of both worlds.

+ Clodo vs building on Cloudflare Workers directly?

Clodo is built on top of Cloudflare Workers. It provides higher-level abstractions, service templates, and middleware. If you prefer lower-level control, you can use Cloudflare Workers directly.

Community & Support

+ Where can I get help?

Join our Discord community, check GitHub discussions, or open an issue on GitHub. You can also check the documentation at clodo.dev

+ Is there a Clodo community?

Yes! We have an active community on Discord. Share your projects, ask questions, and learn from other developers.

+ Can I contribute to Clodo?

Absolutely! Clodo is open source. Check CONTRIBUTING.md on GitHub for guidelines. We welcome bug reports, feature requests, and pull requests.

+ Does Clodo provide enterprise support?

Yes. For enterprise deployments, contact us at sales@clodo.dev for custom support plans.
