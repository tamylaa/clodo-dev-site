Limitations to Be Aware Of

30 Second Timeout

Each request has a maximum 30-second timeout. Use Durable Objects for longer operations.

128MB Script Size

Your uploaded script (including dependencies) has a 128MB limit. Optimize bundle size.

Memory per Isolate

Each isolate instance has memory constraints. Design for stateless operation.

Node.js Compatibility

Workers use V8 isolates, not Node.js. Some npm packages may not be compatible.

No Persistent Storage (Within Request)

Use KV Store or Durable Objects for persistence. Don't rely on local file system.

Billing Concerns

High CPU-intensive operations can get expensive quickly. Monitor and optimize.
