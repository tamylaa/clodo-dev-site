# Debug & Test Files

This directory contains debug scripts, test utilities, and temporary files used during development and troubleshooting.

## Categories

### Debug Scripts
- `debug-build-corruption.js` - Debug build corruption issues
- `debug-emoji.js` - Debug emoji rendering issues
- `debug-full-build.js` - Debug full build process
- `debug-path-adjust.js` - Debug path adjustment logic
- `debug-ssi-only.html` - Debug SSI-only HTML
- `debug-ssi-target.js` - Debug SSI target JavaScript

### Fix Scripts
- `fix-emoji.mjs` - Fix emoji issues (ESM)
- `fix-emojis.js` - Fix emoji issues
- `fix-emojis.ps1` - Fix emoji issues (PowerShell)

### Test Files
- `test-build-process.js` - Test build process
- `test-defer-css-sim.html` - CSS defer simulation test
- `test-defer-css.html` - CSS defer test
- `test-emoji.js` - Test emoji functionality
- `test-exact-build.html` - Exact build test HTML
- `test-exact-build.js` - Exact build test JavaScript
- `test-fix-links.js` - Test link fixing functionality
- `test-full-result.html` - Full result test HTML
- `test-output.html` - Test output HTML
- `test-result.html` - Test result HTML

### Temporary Files
- `tmp.html` - Temporary HTML file
- `tmp_homepage.html` - Temporary homepage HTML
- `temp_lh.html` - Temporary Lighthouse HTML

### Validation Scripts
- `validate-schemas.js` - Schema validation utility

## Usage

These files are typically used for:
- Debugging build issues
- Testing new features
- Validating functionality
- Temporary development work
