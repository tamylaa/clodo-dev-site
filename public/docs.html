Getting Started

Installation

npm install -g @tamyla/clodo-framework

First Service Creation

npx @tamyla/clodo-framework clodo-service create my-first-service --type data-service

Deployment

cd my-first-service
npm run deploy

Core Concepts

Service Types

  • data-service: Database CRUD operations
  • auth-service: Authentication and authorization
  • api-gateway: Routing and API management
  • content-service: Content management
  • generic: Custom service template
