Getting Started with Rails on Workers

1. Create a new Rails Worker project

npx @tamyla/clodo-framework create rails-app my-rails-worker

2. Define your Rails routes

routes.rb
Rails.application.routes.draw do
  resources :posts
  get 'api/users/:id', to: 'api/users#show'
end

3. Create controllers with edge optimization

class PostsController < ApplicationController
  def index
    @posts = Post.all
    render json: @posts
  end

  def create
    @post = Post.new(post_params)
    if @post.save
      render json: @post, status: :created
    else
      render json: @post.errors, status: :unprocessable_entity
    end
  end
end

Database Integration

Clodo Framework provides seamless integration with Cloudflare D1 and other databases for your Rails applications.

D1 Database Configuration

config/database.yml
production:
  adapter: d1
  database: <%= ENV['DATABASE_URL'] %>
  pool: 5
  timeout: 5000

ActiveRecord Models

class Post < ApplicationRecord
  validates :title, presence: true
  validates :content, presence: true

  scope :published, -> { where(published: true) }
  scope :recent, -> { order(created_at: :desc) }
end
