Erste Schritte mit Clodo Framework

Bereit, produktionsreife Workers-Anwendungen zu erstellen? So starten Sie mit Clodo:

Schnelles Setup

npm create clodo@latest mein-app
cd mein-app
npm install
npm run dev

1. Erstellen Sie Ihr Projekt

Verwenden Sie unsere CLI, um ein neues Clodo-Projekt mit allen Enterprise-Funktionen vorkonfiguriert zu scaffolden.

2. Konfigurieren Sie die Umgebung

Richten Sie Ihr Cloudflare-Konto, D1-Datenbank und KV-Namespaces über unser geführtes Setup ein.

3. Deployen Sie in die Produktion

Ein-Kommando-Deployment mit automatischer Skalierung, Überwachung und Sicherheit.

Workers Boilerplate Bewährte Praktiken

Wählen Sie basierend auf Projektanforderungen

Für einfache APIs oder geplante Aufgaben reichen grundlegende Vorlagen aus. Für komplexe Anwendungen, die Authentifizierung, Datenbanken und Multi-Tenancy erfordern, wählen Sie Frameworks wie Clodo, die diese Funktionen von Haus aus bieten.

Berücksichtigen Sie langfristige Wartung

Bewerten Sie Community-Größe, Dokumentationsqualität und Update-Häufigkeit. Offizielle Cloudflare-Vorlagen bieten Stabilität, während Community-Frameworks Innovation bieten.

Starten Sie klein, skalieren Sie nach oben

Beginnen Sie mit einer minimalen Vorlage und fügen Sie Funktionen nach Bedarf hinzu. Die meisten Boilerplates unterstützen inkrementelle Verbesserungen ohne komplette Neuschreibungen.
